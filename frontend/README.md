# Frontend — SGBD (React + Vite)

Interfaz de usuario construida con **React 18** y **Vite** como bundler.

## Estructura

- `src/components/` — componentes reutilizables de UI
- `src/pages/` — vistas/páginas de la aplicación
- `src/routes/` — configuración de enrutamiento (React Router)
- `src/services/` — llamadas a la API del backend
- `src/hooks/` — hooks personalizados
- `src/context/` — manejo de estado global (ej. sesión/autenticación)
- `src/assets/` — imágenes, íconos, estilos
