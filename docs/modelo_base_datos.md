
Este documento técnico describe la estructura relacional y el diseño del modelo de datos implementado para el control de inventario, usuarios y préstamos de la biblioteca.

* **Usuarios:** Registro de alumnos, docentes y personal externo con acceso al sistema de préstamos.
* **Libros (Inventario):** Catálogo completo de títulos, autores, editoriales y ejemplares disponibles.
* **Préstamos:** Gestión de transacciones, fechas de salida, fechas de devolución y estado de los libros.
