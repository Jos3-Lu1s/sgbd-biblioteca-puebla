# Sistema de Gestión de Biblioteca Digital (SGBD)

Plataforma web para que instituciones públicas y educativas administren catálogos
digitales, gestionen préstamos virtuales, controlen usuarios y generen reportes.

## Stack tecnológico

- **Backend:** Node.js + Express (arquitectura MVC), Prisma ORM, PostgreSQL, JWT
- **Frontend:** React 18 + Vite
- **Metodología:** Scrum

## Estructura del repositorio

- `backend/` — API REST bajo arquitectura MVC (modelos, controladores, rutas, servicios)
- `frontend/` — SPA en React (componentes, páginas, rutas, servicios)
- `docs/` — Caso de estudio, justificación arquitectónica y metodológica
